# Contributing to Gemini Code Fixer

Thank you for considering contributing! We love pull requests.

## How to Contribute

1.  **Fork the repository** and clone it to your local machine.
2.  **Create a branch** for your edit: `git checkout -b name-of-your-branch`.
3.  **Make your changes**.
4.  **Test your changes**: Load the extension in your browser and ensure Gemini still works correctly.
5.  **Run Prettier**: Ensure code formatting is consistent.
6.  **Push** to your fork and **submit a Pull Request**.

## Adding New Themes

1.  Download the `.min.css` file from [highlightjs.org](https://highlightjs.org/download).
2.  Place it in the `/themes` folder.
3.  Add the option to `popup.html`.

## Code of Conduct

Please be respectful and professional in issues and pull requests.
