# Privacy Policy for Gemini Code Fixer

**Effective Date:** 07.01.2026

This privacy policy governs the use of the "Gemini Code Fixer" browser extension.

## Data Collection
The "Gemini Code Fixer" extension **does not collect, transmit, or store any user data**.

* **No Analytics:** We do not track your usage.
* **No Server:** The extension runs entirely locally on your device.
* **Settings:** Your preferences (theme selection, on/off state) are stored locally in your browser's storage (`chrome.storage.local`) and are never sent to any external server.

## Code Processing
The extension reads the code blocks generated by Google Gemini on `gemini.google.com` solely for the purpose of applying syntax highlighting styles. The content of your chats is processed locally within your browser and is not read or stored by the extension developer.

## Contact
If you have questions about this policy, please open an issue on our [GitHub repository](https://github.com/jasperfue/Gemini-Code-Fixer)
